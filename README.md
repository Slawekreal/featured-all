# featured-all
Dieses WordPress-Plugin ermöglicht es, das Beitragsbild einzelner Beiträge optional durch ein „Featured Video“ (selbst gehostet oder von YouTube) zu ersetzen. Dazu gibt es eine Metabox im Editor und eine Theme-kompatible Ausgabe. Selbstverständlich können Sie mit diesem Plugin auch weiterhin ein Feature-Bild nutzen.
